# math-candidate-job-similarity
Mathematical analysis of candidate-job similarity using vector space models, cosine similarity and Python.

## Project Overview

This project explores how candidate profiles and job descriptions can be represented as vectors in a multidimensional space. Using concepts from linear algebra, the project analyzes similarity measures such as cosine similarity and Euclidean distance.

The goal is to demonstrate how mathematical models can be applied to real-world recruitment problems using Python.

## Methods Used

The project applies several mathematical and computational concepts from linear algebra.

The similarity measures analyzed include:

- Euclidean Distance
- Dot Product
- Cosine Similarity

These methods are implemented in Python and demonstrated through simple vector examples.

## Project Structure

The repository contains the following components:

- **Final Exam Project.ipynb** – the main Jupyter notebook containing the mathematical analysis, explanations and Python implementation.
- **README.md** – project description and documentation.
- **LICENSE** – MIT license for open-source usage.

The notebook combines theoretical explanations, mathematical formulas and Python experiments to illustrate similarity measures in vector spaces.

## Technologies Used

This project was implemented using the following tools:
